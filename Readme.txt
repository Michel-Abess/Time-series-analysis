==============================================================
Prédire la demande d'électricité
==============================================================

l'objectif de ces travaux est d'aider un acteur du secteur de l'énergie à prévoir le niveau nécessaire de production d'électricité dans un contexte où la demande en électricité des utilisateurs varie au cours du temps, et dépend de paramètre comme la météo (température, luminosité, etc.), le challenge ici est de mettre en adéquation l'offre et la demande en électricité.


===============================================================
Files
===============================================================

	- données_mensuel_conso : données mensuelles contenant les informations sur la consommation électrique en France. Record : 1523
	- dju.xlsx : degeneration jour unifié consommation chauffage électrique à Paris. Record: 118
	- Readme.txt
	- P9_02_graphiques :  Contient l'ensemble des graphiques utiles pour l'analyse

	
===============================================================
How to open file : demande_electricite.ipynb in jupyter notebook
================================================================

S'assurer que le chemin d'accès aux fichiers est bien spécifié pour que le notebook puisse charger les données. Les fichiers peuvent être placés à la racine du dossier(Au même niveau que /demande_electricite.ipynb).

Open terminal and use the following command line:
	- ipython notebook detection de demande_electricite.ipynb
 	- jupyter notebook detection de demande_electricite.ipynb
If you haven't installed python yet, you can use the command line: pip install ipython
